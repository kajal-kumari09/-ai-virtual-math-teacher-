# AI Virtual Math Teacher 🤖

## Description
This is an AI-based web application that solves basic math problems and displays results in a chat format.

## Features
- Solve math problems
- Chat interface
- Instant answers
- User-friendly design

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run

2. Open index.html in browser
